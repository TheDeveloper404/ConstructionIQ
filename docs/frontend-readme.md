# Frontend - Ghid rapid (versiune RO)

**Versiune document:** RO (Romana)

Acest proiect frontend a fost initial generat cu [Create React App](https://github.com/facebook/create-react-app).

## Scripturi disponibile

In directorul proiectului frontend poti rula:

### `npm start`

Porneste aplicatia in modul de dezvoltare.
Acceseaza [http://localhost:3000](http://localhost:3000) in browser.

Pagina se reincarca automat la modificari, iar erorile de lint apar in consola.

### `npm test`

Porneste runner-ul de teste in modul interactiv (watch).
Detalii: [running tests](https://facebook.github.io/create-react-app/docs/running-tests).

### `npm run build`

Genereaza build-ul de productie in folderul `build`.
Build-ul optimizeaza bundle-ul pentru performanta si genereaza fisiere minificate cu hash-uri in nume.

Detalii: [deployment](https://facebook.github.io/create-react-app/docs/deployment).

### `npm run eject`

**Atentie: operatiune ireversibila. Dupa `eject`, nu poti reveni.**

Comanda muta configuratiile (webpack, Babel, ESLint etc.) direct in proiect pentru control complet.
